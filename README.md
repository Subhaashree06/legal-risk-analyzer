# ⚖️ Legal Risk Analyzer

## 📌 Project Description

The **Legal Risk Analyzer** is a web-based application that analyzes legal documents and classifies clauses based on their risk level (High, Medium, Low). It helps users quickly identify potentially risky statements in contracts or agreements.

---

## 🚀 Features

* 📄 Upload legal documents (PDF/Text)
* 🔍 Automatic clause extraction
* 🤖 Risk classification (High / Medium / Low)
* 📊 Visual representation using charts
* ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Libraries:**

  * Flask
  * PDF processing libraries
  * Machine Learning model (for classification)

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/Subhaashree06/legal-risk-analyzer.git
cd legal-risk-analyzer
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate   (Windows)
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the application

```
python app.py
```

### 5. Open in browser

```
http://127.0.0.1:5000/
```
